# weapp-vite-tailwindcss-template

`weapp-vite` 集成 `tailwindcss` 的模板

## 使用方式

### 开发

- `pnpm dev`

- `pnpm dev --open` 可以打包并直接启动微信开发者工具

### 构建

`pnpm build`

### 打开微信开发者工具

`pnpm open`

### 生成组件/页面

`pnpm g path/to/your/component`
